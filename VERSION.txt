Version: 2.0.0
Release type: Full application architecture and UI rebuild

Highlights:
- Replaced the old zoom-based responsive logic with a viewport-safe app shell
- Added a desktop chapter rail and synchronized current-scene chrome
- Rebuilt all scene layouts, cards, charts, controls, overlays, and presenter views
- Added isolated per-scene scrolling so content never sits under the header or footer
- Added compact portrait and landscape layouts without hiding scientific content
- Refactored overlays so Notes, Presenter, and Research panels cannot overlap
- Added deterministic experiment particles and cancel-safe animation timers
- Added touch swipe navigation with vertical-scroll protection
- Improved keyboard focus, ARIA state, reduced-motion support, and dialog fallback
- Updated the PWA cache and manifest to v2.0.0
