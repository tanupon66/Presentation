Version: 1.3.1
Release type: Mobile UI overlap hotfix + cache update

Fixes:
- Rebuilt the mobile page shell so the header no longer wraps over slide content
- Utility buttons are hidden from the mobile header and moved to the popup command menu
- Each slide now scrolls inside a fixed stage in portrait mode instead of extending under the header/footer
- Compact landscape mode scales the active slide to the available screen with a lower safe scale limit
- Presenter and notes overlays are non-interactive and invisible until explicitly opened
- Core HTML/CSS/JS now use network-first service-worker handling and versioned asset URLs to reduce stale UI after deployment
